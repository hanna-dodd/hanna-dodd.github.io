Art Intelligence: Unpaired Image Translation

Cycle GAN that generates artwork based on landscape photographs

Our team developed a cycle generative adversial network to create artwork out of photographs. Using a dataset of paintings done by Paul Cezanne and a dataset of landscape photographs, we are able to translate between the two using unpaired image translation. This project is able to generate artwork to a similar degree of a human artist.

Art Intelligence was submitted at CUCAI 2021 and was published in the conference proceedings here

MNIST Digit Recognition

CNN to Recognize Handwritten Digits

We created a convolutional neural network to identify handwritten digits from 0-9. We acheived a validation accuracy of 98.17% and deployed an interactive application to demonstrate our project. This project was completed within Western AI's Gideon Stream and won first place at the Western AI Project Showcase.

January 2021

PiWriting

Accessibility in an Online Learning Environment

PiWriting is able to take an image of a handwritten sentence or message and convert it to digital text form. The text can then be read aloud or saved as a separate file for further use. Our goal was to make this system as easy to use as possible so that it can be useful for those with visual impairments or disabilities that affect a person's ability to read or take notes.
